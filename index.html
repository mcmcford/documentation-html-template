Introduction

Hey! Welcome to QuoteBots public facing documentation, hopefully you'll find everything you need in the lines below =)

This documentation aims to provide a detailed description and example on how to use each of the commands QuoteBot provides it's users with, if there is any errors, something missing or you simply want to say hey, feel free to join the discord.

Getting Started

When first inviting QuoteBot to your server, you will be greeted with the following welcome message:

This is simply a friendly greeting and the first set in getting quotebot up and running!

The !setquotechannel command simply sets the channel where you want every quote to be sent and can be used as follows: !setquotechannel #inspirational-quotes.

Other commands you might find useful are !setbotchannel and !restrictcommands which you can read more about in the commands section of this website.

Commands!

General Commands -

!quote - This is obviously one of the more important commands that exists, this is used for quoting someone within the server, it is fairly simple to use as can be seen in these examples:

!quote @josh "I like this bot"
!quote @martin *clash of clans music*

Simple right?
Well, if you want to take this a step further, the bot will also accept the users DiscordID opposed to a ping, meaning you can quote them without needing to ping them as can be seen below:

!quote 462300281817006111 "I like this bot"
!quote 144194743432118277 *clash of clans music*



!quote2 - This command is similar to the previous command however it allows you to quote an interaction between two people in the server. Think of it like a question response format (I am currently working on expanding this to allow for more than 2 inputs / quotes). You can see how it would be used in the examples below:

!quote2 @max "Jacob someone made a website..." @jacob "NO!"
!quote2 @tom "smh" @josh "SHUT THE F*** UP TOM"

What if you want to quote two people saying the same thing?
Well, there is an easy way to do that too!

!quote2 @tom @harry "Jinx!"



!outsidequote - (!oquote for short) What to quote something you overheared in your day to day life, or a family member who doesn't have discord but you really need your friends to know what they said? then you can use the !oquote command (short for !outsidequote). This allows you to quote non-discord users as you can see in the examples below:

!oquote [My Mum] "I'm a cat girl"
!oquote [Average overwatch player] "When in doubt bastion out!"



!customquote - (!cquote for short) This is a much more flexible command which can do everything !quote, !quote2 and !oquote can do alongside a bunch more (like adding images to quotes!). So! If you want to quote more than two people, want to add context to a quote with an image or want to be able to quote more than one 'outside' user / non-discord user then this command will allow you do it all. It is a little more complicated to use however so all the details have been carefully layed out below:

!customquote take a series of arguments as listed below:
-[u] (for user) This accepts any @mention of a user or any valid userID
-[o] (for outside user) This accepts a string value so 'John Doe', 'Alice', 'Henry Tudor, son of Henry VII and Elizabeth York' are all valid names
-[q] (for quote) This, once again, is just a string value so: "Funny thing" would be valid
-[i] (for image) This accepts a URL of an image eg "https://placekitten.com/g/200/300"


So, to bring that all together, the commands for !customquote would look something like this:
!customquote -[u] @morgan -[q] "Funny thing morgan said" -[u] @Harry -[o] Harrys friend who isn't on discord -[q] *funny thing harry and his friend said simultaneously* -[i] https://relaventimagetothestory.com/image.jpeg

It is quite long-winded however currently it seems like the best way to tackle 'custom quotes' and shouldn't be too hard to pick up. There are various combinations of arguments you can you, you just need to make sure that you stick to the guidelines below:
Don't quote more than 25 people
Only use 1 image
Keep all quotes below 256 characters long



!edit - [This currently only works with !quote created quotes, not !quote2 or !oquote (they are still in development)] Everyone is prone to making a typo every now and again so the !edit command allows you to fix any errors you make. Using this command you can edit either the quote it's self, or the person you were quoting by following this format:

!edit {quoteID} quote "Hey, this is the version without the typo!"
or
!edit {quoteID} person @ThePersonWhoActuallySaidTheFunnyThing

For this, i should clarify, the 'quoteID' is the discord Message's ID, if you don't know how to find that, you can read the steps in this offical discord post. It is also important to remember, every discord message has a different ID so it is important to use the original ID of the quote (the first time the quote was posted, usually in the servers dedicated quote channel) as the IDs of any other copies of the quote, such as those that may come up when using the !randomquote, !quotesearch or !getquote command won't work.
You can see full examples of how the quote should work below:

!edit 918894597797724270 quote "Hahaha"
!edit 918893635662147594 person @max



!random - This command is very simple, it just finds and posts a random quote from the server (if you are going through a lot of random ones, you can shorten the command to !r although this command does have a short cooldown)



!quotesearch - This command is used for searching for quotes that contain certain words or phrases, you can see how they would be used below:

!quotesearch "Jinx"
!quotesearch f***

Both of these will return any quotes that contain those words / phrases.



!searchdate - This command is used for searching for quotes made on a specific day, following the format shown below:

!searchdate 15-10-2021
!searchdate 09-07-2020

Both of these will return any quotes that were made on the respective dates.



!getquote - If you ever want a random quote from a specific person, this is the command for you, it is as simple as mentioning them after the command and a random one of their quotes will be posted, some examples are shown below:

!getquote @Max
!getquote 839902058362699806



!getquotes - This command is almost identical to the command above however this one can get multiple quotes from a person. It is used as demonstrated below:

!getquote @Max 10
!getquote 839902058362699806 15

These will both return 10 and 15 quotes from the specified person respectfully (assuming they have that many quotes)



!quotestats - Ever wondered how many times you have been quoted or how many quotes you have made? well, this command can find that for you, as well as the stats of others!

!quotestats
!quotestats @Abby

The first example above will return your quote stats, the second will return the stats of the person mentioned (using their discord ID will also work rather than pinging them)



!guess - This command will bring up one of 3 games assuming they're enabled in the server, these games often include guessing who said or who didn't say a certain quote.



!gamestats - Similar to quotestats, this will give you information about your stats in relation to the !guess games, see the examples below for how to use it:

!gamestats
!gamestats @Martin

The first example above will return your game stats, the second will return the stats of the person mentioned (using their discord ID will also work rather than pinging them)



!starwarsquote - This is as simple as it sounds, it will give you a random starwars quote since ... why not?



!gameleaderboard - This is another simple command, all this one does is show a leaderboard of the people in your server in relation to the games



!guesses - This command simply shows how may times any of the !guess games have been played within your server



!mydata - [This command is still being tested] This command will collect all the data QuoteBots database stores about you, and it will DM it to you in the form of a HTML document



!delete - [This command is still being developed] This command will allow you to delete / anonymise all data relating to you that is stored in QuoteBots database, this includes deleting quotes made about you, outsider quotes you have made, your scores, any of your data stored in the databases cache, alongside anonymising any quotes you made.



!uptime - This will simply show you both the bot and the servers uptime.



!ping - This will show you the bots ping.



!myvote - This command will show you all of your votes for the QOTY nominations (if a QOTY event is currently happening)



!meta - This command will let you see various details about a quote, it is best used with the message ID of the quote (in servers with legacy quotes this may not always work). You can see examples of how to use it below:

!meta 914365674602766346
!meta 926639487663108126



!qoty - This command is used to cast your vote during a QOTY event. It is followed by either an @mention of a user or the message ID of a quote from the previous year that you would like to nominate as shown below:

!qoty 914365674602761029
!qoty 926619487663107384



Admin Commands -

!setquotechannel - This is used to set which channel you would like any quotes that are made to be sent and stored in (I suggest having a dedicated channel for this). You can see how it is used below:

!setquotechannel 878028210602922005
!setquotechannel #inspirational-quotes

As you can see, both the # for the channel and its ID can be used.



!setbotchannel - The bot channel is used if you want to restrict bot commands such as !guess, !quotesearch and !getquotes to a bot channel, this doesn't affect the commands used to create a quote as the bot quickly deletes the message and any messages it sends as to prevent spam.

!setquotechannel 878028210602924730
!setbotchannel #bot-spam



!restrictcommands - [This command is going to be updated soon] This is used in conjunction with !setbotchannel to toggle whether the commands considered more 'spammy' should be restricted to the bot channel or not, it can be used as shown below

!restrictcommands true
!restrictcommands false



!config - [This command is going to be updated soon] This command with show you all the current configurations for your server, from members blacklised from certain command outputs (see below) to what channel is set as your quote channel etc.



!addrandomblacklist - This command is used to stop a quotes made about a specific user from showing up when anyone uses !random. You may find this useful if there was a falling out within the group or if you think someones quotes might be a little too inappropriate to have coming up again and again. See how to use this command below:

!addrandomblacklist 209026834115985407
!addrandomblacklist @Max



!removerandomblacklist - This is used for removing someone from the random quotes blacklist that was described above and is used in the exact same way, see below:

!removerandomblacklist 209026834115985407
!removerandomblacklist @Max



!addgamesblacklist - This command stops quotes of a certain user from appearing in the !guess games, this may be useful if you haven't disabled the users ability to quote bots and simply don't want bots quotes from being used in the games, similarly this can be used to stop specific users quotes from appearing, see how to use it below:

!addgamesblacklist 672554834115985407
!addgamesblacklist @QuoteBot



!removegamesblacklist - This is used for removing someone from the !guess games blacklist meaning there quotes may come up in any of the games again.

!removegamesblacklist 672554834115985407
!removegamesblacklist @QuoteBot



!addgamethreeblacklist - This command is very specific but it can be useful ~ the stops certain users from appearing in the third game that !guess may bring up. This game is focused on guessing who joined the discord server first. So you obviously might want to disable the servers owner as they're usually always the person who joined first. you can see how to use it below:

!addgamethreeblacklist 56478512354987425
!addgamethreeblacklist @Charlie



!removegamethreeblacklist - This one is fairly self-explanatory, it removes people from the blacklist described in the command above.

!removegamethreeblacklist 56478512354987425
!removegamethreeblacklist @Charlie



!enablegameone - If you want to disable game one from coming up in your server, this command can help you do it! Game one is also known as "Guess who said the quote ...".



!enablegametwo - If you want to disable game two from coming up in your server, you can do it with this command. Game two is also known as "Guess which of the quotes wasn't said by ...".



!enablegamethree - If you want to disable game three from coming up in your server, you can use this command. Game three is also known as "Guess who joined the server first". The main reason someone may want to disable this game is if everyone joined the server at very similar times .



!enablecensor - [This command is going to be updated soon] This command will hide the creator of a quote when it appears in the !random, !getquote(s) or !quotesearch commands if the creator is on the random quote blacklist. You can see how to use this command below:

!enablecensor true
!enablecensor false



!enablebotquoting - [This command is going to be updated soon] This command simple lets you stop or allow people to quote bots opposed to actual users.

!enablebotquoting true
!enablebotquoting false



!qotyrestrict - Using this command will bring up a selection box allowing you to restrict people in the General Blacklist (previously known as the 'random blacklist') from participating in QOTY. Quotes of them will not be able to be used in QOTY, nor will they be able to participate.



!startqoty - [This command is going to be updated soon] This command is used to start QOTY in your server, carrying out a series of actions such as creating a channel dedicated to QOTY and explaining how users can participate in QOTY

What is QOTY?

Quote Of The Year (abbreviated to QOTY) is an event that each server has the option of running from the first of december (GMT) through to the last day of january where users within the server can vote for quotes made between january and december to decide which deserves the award QOTY amongst other titles. Alongside this various stats are shown such as most quoted, most quotes made and the most common words used in quotes etc.
